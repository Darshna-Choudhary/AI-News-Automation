# AI News Automation System

## Overview
An automated news delivery system built using n8n, Gemini AI,
Google Sheets, and Telegram.

## Features
- Automated news collection
- AI-powered summarization
- Scheduled execution
- Telegram delivery

## Tech Stack
- n8n
- Gemini API
- Google Sheets
- Telegram Bot
- Render

## Workflow
News Sources
↓
n8n
↓
Gemini AI
↓
Google Sheets
↓
Telegram
